About this visualization

This interactive network shows hip-hop artists and their collaborations. Each node represents an artist and each edge represents a collaboration.

How to use

Use Search to find an artist by name. Click a node to open the information pane with details and connections. Use Group Selector to highlight communities. Use the zoom controls to navigate.

Visual encodings

Node size: centrality (more connected / more central artists appear larger).
Node color: community / cluster (groups detected by the layout/partition).

Data & methodology

Built in R (data cleaning + network construction), exported to Gephi and published using the Sigma.js web export. The dataset reflects the source used in the project and may be incomplete or time-dependent.
